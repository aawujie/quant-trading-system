# SSH 端口转发访问指南

## 建立SSH隧道

在你的本地电脑执行以下命令：

```bash
ssh -L 8080:localhost:8080 ubuntu@43.133.40.16
```

### 命令说明

- `-L 8080:localhost:8080`: 本地端口转发，将本地的8080端口映射到远程服务器的localhost:8080
- `ubuntu@43.133.40.16`: 远程服务器的用户名和IP地址

## 访问服务

建立隧道后，在浏览器中访问：

```
http://localhost:8080
```

此时你访问的实际上是远程服务器43.133.40.16上运行在8080端口的服务。

## 注意事项

- SSH隧道需要保持连接状态才能继续访问
- 关闭SSH连接后，端口转发也会断开
- 如果本地8080端口已被占用，可以修改本地端口号，例如：`ssh -L 8888:localhost:8080 ubuntu@43.133.40.16`
